# many-vid-to-1-audio
folder full of video -> all audio + 1 audio with embedded metadeta
---
# steps
1.install req :
    ``pip3 install -r requirements.txt && sudo apt install ffmpeg ``
2.creat virtual env
  ``python3 -m venv venv``
3.enter virtual env 
  windows:``venv\Scripts\activate``
  linux/mac:``source venv/bin/activate``

4.run: 
  ``python3 main.py``
